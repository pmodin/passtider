# passtider.py

Displays available times in your terminal.

Currently only Sthlm City, you need to change the code accordinly for other
places.

Example usage (pipenv)

```
# install dependencies 
$ pipenv install

# run main.py
$ pipenv run python main.py
2022-07-27 14:30:00
2022-07-29 09:20:00
```
